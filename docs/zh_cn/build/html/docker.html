Docker¶

Gerapy 同样提供了 Docker 镜像，可以使用 Docker 快速启动 Gerapy 服务。

运行¶

首先需要选定一个目录作为 Gerapy 工作目录，如 ~/gerapy 或其他，在这里以 ~/gerapy 目录为例，Gerapy 启动命令如下：

docker run -d --name gerapy -v ~/gerapy:/app/gerapy -p 8000:8000 germey/gerapy

运行之后，直接访问 http://localhost:8000 即可进入 Gerapy。

默认的登录用户名为 admin，密码为 admin，用户名密码可以在 http://localhost:8000/admin/auth 自行修改。

其中命令参数说明如下：

  • -d：后台运行

  • –name：指定容器名称

  • -v：指定挂载目录

  • -p：指定绑定端口，前者为宿主机端口，后者为容器端口

源镜像¶

本镜像 Registry 在 Docker Hub，其地址为：Gerapy，其版本与 Gerapy 一一对应，版本列表见：Tags。
